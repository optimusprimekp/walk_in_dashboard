Build a complete enterprise-grade Walk-In Interview Management System similar to Passport Seva Kendra token management.

Technology Stack
Frontend: React.js + TypeScript + Tailwind CSS + Shadcn UI
Backend: Node.js + Express.js
Database: PostgreSQL
Realtime: Socket.IO
Authentication: JWT
TV Display: Real-time Dashboard
Excel Import: XLSX Support
Deployment: Docker + Nginx
System Overview

The system manages walk-in interviews using QR code check-in, token generation, interviewer assignment, waiting queue management, and interview timing.

The process should work exactly like Passport Office token management.
